# E-Commerce Website

A Website made with React.js Framework.




## Features

- Fully Responsive
- AI suggestion.






## Run Locally

Clone the project

```bash
  git clone https://github.com/saraelmasryy/re-fashion
```

Go to the project directory

```bash
  cd re-fashion
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
